## What session tool app does

session tool app is a compact web trading helper built around session window review and Pre London Session planning. It gives traders one place to study market timing, read session behavior, and shape a working bias before the next active period starts.

Rather than covering charts, execution, or portfolio management, the app stays centered on session prep. If you want a focused tool for examining trading sessions and organizing your view before the market moves into a new window, this project keeps the process tight and practical.

## How to use it

Open the app before your trading session starts, then use it to check the current window, review Pre London conditions, and record your session bias.

Typical workflow:
1. Load the app.
2. Review the relevant session window.
3. Note the current market context.
4. Set or update your session bias.
5. Recheck the setup as the session develops.

Treat it as a preparation layer, not a complete trading platform. The purpose is to keep session reading clear, structured, and easy to repeat.
